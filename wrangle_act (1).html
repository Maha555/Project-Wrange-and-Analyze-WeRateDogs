Programmatic Assessment¶

Tidiness Issues¶

  • Merging the three dataframes, using the common tweet_id
  • The puppo/doggo/floofer/pupper terms in df_archive table should be merged into one column named "dog_stage"

Quality Issues¶

  • Some columns contain information that will not be needed for my analysis
  • Tweet_id in df_archive table should be a string, not integer
  • timestamp column in df_archive table should be converted to dateTime
  • img_num data type in df_archive table should be a category type, not float
  • Some tweets in expanded_urls do not contain an image
  • Some of the numbers from rating denominator not equal 10
  • Some of the numbers from rating_numerator
  • Some dog names are not actually names
